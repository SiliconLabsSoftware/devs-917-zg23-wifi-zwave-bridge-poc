typedef int(* ZW_SendRequst_Callback_t) (BYTE txStatus, BYTE rxStatus, ZW_APPLICATION_TX_BUFFER *pCmd, WORD cmdLength, void *user)

Callback to sl_zw_send_request.

Parameters
txStatussee ZW_SendData
rxStatussee SerialAPI_Callbacks::ApplicationCommandHandler
pCmdpointer to received data, NULL if no response was received.
cmdLengthlength of received data 0, if no response was received.
userUser defined argument, send with sl_zw_send_request
Returns
True if more responses are expected. False if none is expected For example, a multi channel endpoint find expects more multi channel find report frames in response if the device has a big number of endpoints which cannot fit in on the Z-wave frame.
