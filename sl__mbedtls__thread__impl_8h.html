void sl_mbedtls_threading_mutex_init ( void )

Initialize a mbedTLS mutex using CMSIS-RTOS2.

This function creates a new CMSIS-RTOS2 mutex and assigns it to the mbedtls_threading_mutex_t structure for use by mbedTLS.
