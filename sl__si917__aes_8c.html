Decrypt data using AES-128 ECB mode with mbedTLS.

Decrypts the input message using AES-128 in ECB mode via mbedTLS. The input length must be a multiple of 16 bytes.

Parameters
msgPointer to the input data to decrypt.
msg_lenLength of the input data in bytes (must be multiple of 16).
keyPointer to the 128-bit (16-byte) AES key.
ivPointer to the IV (not used in ECB mode, can be NULL).
msg_outPointer to the output buffer for decrypted data.
Returns
0 on success, -1 on failure.
