void sl_config_init ( void )

Initialize the Z-Wave gateway configuration with default values.

This function initializes the router_cfg structure with default values including network addresses, security settings, and device parameters. It sets up IPv6 addresses for the gateway, LAN, PAN, and tunneling interfaces.
