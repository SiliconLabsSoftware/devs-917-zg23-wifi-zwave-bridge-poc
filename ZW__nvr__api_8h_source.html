71/*=============================== NVRGetValue ===========================
72** Get a value from the NVR flash page.
73**
74** Offset 0 is the first byte in the protocol NVR area.
75**
76** Offset from NVR_APP_START_ADDRESS to NVR_APP_END_ADDRESS
77** are NVR memory reserved for the application
78**
79** If the CRC16 field in the protocol area of the NVR is not correct
80** all fields will return the value 0xFF when read with this function.
81**
82** Side effects: None
83**
84**--------------------------------------------------------------------------*/

86ZW_NVRGetValue(BYTE bOffset, BYTE bLength, BYTE *bRetBuffer);
