Add an extended header option to a frame

Parameters
pktPointer to ZIP_PACKET
bufsizeSize of buffer containing ZIP_PACKET including variable length payload buffer. The actual space used is encoded in headers in the payload as per this function.
opt_typeType of extended header option to add.
opt_valValue for new extended header option (does not include type and length)
opt_lenLength of value for new extended header option.
opt_len_sizeNumber of bytes in the option header to use for the length. Must be 1 or 2 bytes.
Returns
Number of bytes added to the packet. 0 if bufsize is too small to add the provided extended header option.
